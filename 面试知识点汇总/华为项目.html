operation and maintenance unit(OMU) : 操作维护单元。
1）操作维护单元，是一个逻辑概念。在操作维护子系统中，OMU是操作维护终端和主机之间的通信桥梁。 2）CIE（Carrier-grade Infrastructure application Environment）提供给用户的操作界面，对整个系统的软硬件进行统一管理，包括对系统中设备管理、部署等功能的操作维护能力，包括配置、监控、告警、日志等OM能力。
IP multimedia public identity(IMPU): IP多媒体公共标识。公有用户标识（IMPU）是用户在IMS网络中通信的标识，公有用户标识用于SIP消息的路由。一个IMS用户可以分配一个或多个公有用户标识。公有标识是该用户对外可寻址的标识；一个或多个公有标识对应一个业务Profile
IP multimedia private identity(IMPI): IP多媒体私有标识。一个由归属网络运营商定义的具有全球唯一性的身份标识，用于在归属网络中从网络的角度唯一地标识用户的签约关系。私有用户标识（IMPI）用于用户接入IMS网络的注册、鉴权、认证和计费，IMS网络内的私有用户标识应保证唯一性。

session control unit(SCU): 会话控制单元 。CSC3300的进程，主要完成SIP UDP的终结和SIP呼叫控制处理，包括SIP协议的处理、Diameter编解码和协议处理、呼叫连接和业务触发、业务逻辑控制实现、计费等功能。

media gateway controller(MGC): 媒体网关控制器。负责对媒体网关中与媒体通道连接控制相关的呼叫状态部分进行控制的网关控制设备。
media gateway(MGW): 媒体网关。一种核心网设备，它将一种网络的媒体转换成另一种网络所要求的格式，可以分别对音频、视频和数据进行处理，并且能够进行全双工的媒体转换。也可以播放一些音频/视频信号，执行一些IVR功能，甚至具有提供媒体会议的能力。
IP multimedia media gateway (IM-MGW): IP多媒体网关。用于实现IMS和CS（Circuit Switched domain）互通的边缘接入层产品。IM-MGW主要负责IMS域与PSTN、CS域的传统交换方式转换为IP方式，以及必要的Codec编解码变换功能，从而实现全网IP化通信。
Media Gateway Control Function (MGCF): 媒体网关控制功能
Session Initiation Protocol (SIP): 会话发起协议，是一个互联网工程任务组（IETF）标准协议，它用于发起视频、语音、聊天、游戏以及虚拟现实等多媒体元素的交互用户会话。
Core Network (CN): 核心网
Circuit Switched (CS): 电路交换。CS域是指为用户业务与相关信令提供的CS连接所有核心网实体。CS连接是指在连接建立与释放过程中的分配专用网络资源
packet switched(PS): 分组交换。在通信网络发送消息的方法，较长的消息被拆分成较短的报文。报文以消息交换的方式进行发送。
Session Description Protocol (SDP):
call session control function(CSCF): IMS网络的核心部件，主要完成注册、鉴权、会话控制、业务触发、拓扑隐藏、QoS控制、NAT穿透、安全管理等功能。
proxy-call session control function(P-CSCF): 代理-呼叫会话控制功能。 业务应用过程中用户接入IMS网络的第一个接触点。它负责代理所有SIP信令，并完成呼叫的路由控制；提供QoS资源预留；支持SIP信令压缩以提高空中接口的带宽使用效率；提供NAT控制以支持企业网的NAT穿透；同时可以维护与UE之间的安全联盟以保护与UE之间信令的私密性、完整性。
interrogating-call session control function(I-CSCF): 查询-呼叫会话控制功能。 用户归属网络的统一入口点，负责S-CSCF的分配，及被叫所在S-CSCF的查询。同时还完成IMS域间拓扑隐藏功能。
serving-call session control function(S-CSCF) : 服务类型的会话控制功能。 IMS网络的业务交换中心，主要负责接收并处理UE的注册请求、用户管理、会话控制、业务交换、业务控制、SIP消息处理、计费等，并能根据应用的触发原则将SIP请求触发到相应的AS。
session border controller(SBC): 会话边界控制器 。会话边界控制器是为确保VoIP安全性与持续通讯，并提供多媒体服务的，主要安装在边缘接入层的一种边界路由器。
single voice core(SVC): 一张话音核心网。


home subscriber server(HSS): 归属用户服务器。IMS（IP Multimedia Subsystem）用户归属网络中存储用户信息的核心数据库。用于在归属网络中保存IMS用户的签约信息，同时提供管理接口，由运营商及终端用户对签约数据进行定制和修改。HSS中保存的主要信息包括：IMS用户标识、IMS用户安全上下文、IMS用户的路由信息及业务签约信息。类似于GSM系统中的HLR。
Telco Cloud Engine(TCE): TCE。云核心网投资开发的下一代硬件平台，采用4U/8U机框，支持框内计算刀片共享硬件管理、风扇、电源。规划支持NP加速卡、框内交换实现CE/5G媒体面加速。

border control function(BCF): 边界控制功能。从接入网络的广泛性和开放性来看，接入网络被认为是不安全网络。为了保证业务网络的安全和正常运行，IMS网络必须部署在一个相对安全的区域内。 与此同时，在企业用户接入时接入网络中也存在各种各样的NAPT设备，为了识别网络层和应用层地址的区别，也需要在网络的边界层提供相应的NAPT识别功能，保证信令的正常处理。所以在网络的部署中，迫切需要网络的接入边界控制功能，实现接入网和IMS核心网络的完全隔离，同时实现相应的接入边界控制功能，包括安全接入控制、用户接入控制、NAT穿越、QoS管理、媒体管理等功能。
border gateway function(BGF) : 边界网关功能 。IMS组网中设备之一。实现信令网络地址转换，媒体代理，固网接入QoS等功能。BGF作为QoS策略执行点，通过Ia接收来自RM9000的QoS策略配置，完成流分类、CAR功能、流量整形、队列等功能。(SBC媒体处理)
